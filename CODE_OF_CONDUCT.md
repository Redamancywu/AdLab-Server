# Code of Conduct

## Our Pledge

We want this project to be an open, welcoming, and respectful space for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume good intent
- Give feedback in a clear and helpful way
- Focus on improving the project, not attacking people

## Unacceptable Behavior

- Harassment or abusive language
- Personal attacks
- Discriminatory or exclusionary behavior
- Deliberate disruption of discussion or collaboration

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that do not align with this Code of Conduct.

## Scope

This Code of Conduct applies to project spaces, issues, pull requests, and discussions related to this repository.
