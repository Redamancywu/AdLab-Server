DEVELOPER INTEGRATION SDK

简明、内聚的 API 设计

AeroBid 提供高内聚的移动客户端集成 SDK，将繁复的网络通信与上报逻辑封装在库底层，开发者仅需几行代码便可拉起一个轻量变现客户端。

3.2 MB SDK 包二进制体积
三步集成 引入、初始化、调用展示 
import AeroBidSDK

// 1. 初始化 SDK 核心
AeroBidSDK.shared.initialize(
    appKey: "ab_prod_8f9c0e4",
    appSecret: "sec_7d2f9b8c"
) { success in
    print("AeroBid initialize: \\(success)")
}

// 2. 加载广告位对应素材
let banner = AeroBidBanner(placementId: "plc_home_banner")
banner.loadAd { [weak self] result in
    switch result {
    case .success(let ad):
        // 3. 在视口中渲染
        ad.show(in: self?.view)
        print("Ad loaded via: \\(ad.dspName), CPM: \\(ad.cpm)")
    case .failure(let error):
        print("Ad failed to load: \\(error.localizedDescription)")
    }
}

系统物理拓扑

AeroBid 介于客户端、程序化买家与数据库之间，解耦客户端变现逻辑，降低整体维护成本。

1

Client-Side SDK

嵌入终端应用，主要负责广告视口呈现、事件曝光采集、本地数据缓存，并向中间层发起聚合请求。

2

AeroBid Core Server

Go 构建的多协程核心中台。根据配置中心规则执行 Waterfall 轮询，或在 S2S 服务端并发请求下游买家进行高精度比价。

3

DSP & Analytics

向已集成的程序化渠道（DSP）分发 OpenRTB 竞价请求，并异步写入指标与结算数据至本地 SQLite / 外部 PG 数据库。

容器化一键部署

AeroBid 已内置完整的 Docker 容器化配置。你可以直接拉取源码，一键在本地或预发环境运行起完整的变现模拟网络。
